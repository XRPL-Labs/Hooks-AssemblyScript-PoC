# AssemblyScript PoC for XRPL Hooks

## Install

```
git clone https://github.com/XRPL-Labs/xrpld-hooks
cd xrpld-hooks
npm install
cd ..
git clone https://github.com/AssemblyScript/assemblyscript.git
cd assemblyscript
npm install
npm run build
```

## Build & deploy

`./build.sh sSomeSecretKey123123`
